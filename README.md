
# Cameroon Zoom

High-fidelity institutional business directory for Cameroon. Find and contact verified businesses in Douala and Ngaoundéré.

## Core Features
- **Business Discovery**: Expert search with Leaflet mapping and cartoon-style interactive pins.
- **Institutional Governance**: Admin hub for manual document verification (NUI, RCCM, ID).
- **Social Proof**: High-speed "My Stories" (Reviews) with visual photo pickup.
- **Professional Presence**: Standardized business profiles with direct Phone and WhatsApp actions.
- **Offline Reliability**: Browser-level caching for seamless access to previously viewed listings.

## Technical Stack
- **Framework**: Next.js 15 (App Router)
- **Persistence**: Firebase Cloud Firestore
- **Auth**: Firebase Authentication
- **AI**: Genkit with Gemini 2.0 Flash
- **Styling**: Tailwind CSS & ShadCN UI
